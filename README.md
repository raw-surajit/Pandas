# Pandas Mastery: From Basics to Real-World Data Analysis

---

## 📊 Project Highlights

### 1. Anime Data Analysis & Feature Extraction
In this project, I moved beyond simple data manipulation to **Feature Extraction**.
* **Goal:** Extract hidden insights from a raw anime dataset.
* **Key Tasks:**
    * Used custom Python functions and `.apply()` to extract **Episode Counts** from text-based titles.
    * Calculated the **Total Duration** (in months) of anime airtime using date parsing logic.
    * Cleaned and converted string-based data into numerical formats for statistical analysis.
    * Identified the highest-scoring anime and the longest-running series using logic-based filtering.

### 2. Global Countries Insights
A data extraction project focused on handling large-scale datasets with over 60 attributes per country.
* **Goal:** Perform deep-dive analysis on global demographics and governance.
* **Key Tasks:**
    * Found the most and least populated countries using `.max()` and `.min()`.
    * Analyzed **Democratic Scores** to rank top-performing nations.
    * Used **Conditional Selection** to filter countries by region (e.g., Eastern Europe, Africa).
    * Handled **Missing Data (NaN)** to identify countries with unknown political leaders.
